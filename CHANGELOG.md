## [0.1.0] - 2024-01-15

### Added
- Initial project setup and structure
- Docker Compose configuration for local development
- Core project documentation (README, API docs, architecture)
- GitHub Actions CI/CD pipeline
- Base authentication and JWT support
- User registration and login endpoints
- Document upload and management foundation
- KYC verification workflow template
- OCR and face recognition service integration points
- Environment configuration templates
- Contributing guidelines
- Development and deployment guides
